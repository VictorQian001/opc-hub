# Sites

这个目录收录全国 OPC 站点。

建议按区域和城市组织，例如：

```text
sites/
└── china/
    ├── east/
    ├── north/
    ├── south/
    ├── central/
    ├── southwest/
    └── northwest/
```

每个条目建议包含：

- 城市
- 状态
- 公开页面
- 面向人群
- 优惠政策
- 基本介绍
- 信息来源
