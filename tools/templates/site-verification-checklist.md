# 站点核验清单

## 这个工具解决什么问题

用于判断一个 OPC 站点条目是否适合被正式收录，避免目录里出现大量无法核验或信息过时的页面。

## 内容

```text
[ ] 是否有公开可访问链接
[ ] 是否能确认城市
[ ] 是否能确认站点名称
[ ] 是否能确认站点定位或类型
[ ] 是否能找到发布时间或最近一次公开信息
[ ] 是否能标注来源
[ ] 如果信息不完整，是否已标注“待核验”
[ ] 是否避免使用未经公开确认的隐私信息
```

## 使用建议

- 新增站点前先过一遍
- 更新旧条目前再过一遍
- 如果只满足部分条件，可以先放到城市线索页，不一定直接写成已收录站点

## 适用场景

- 站点目录共建
- 城市线索整理
- PR 自查
