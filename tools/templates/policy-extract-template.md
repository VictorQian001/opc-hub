# 城市政策摘录模板

## 这个工具解决什么问题

用于从城市公开政策、新闻通稿或政府页面中，快速摘录与 OPC 相关的有效信息，方便后续整理政策地图或城市页。

## 内容

```text
城市：
页面标题：
发布日期：
来源机构：
原文链接：

与 OPC 相关的核心信息：
1.
2.
3.

涉及的支持内容：
- 空间
- 资金
- 算力
- 人才
- 活动
- 场景

适合落到仓库的页面：
- 城市页
- 站点页
- 政策地图
```

## 使用建议

- 优先摘录政府或官方公开页面
- 保留原始链接
- 用自己的话压缩，不要整段复制原文

## 适用场景

- 做城市页
- 做政策地图
- 整理区域观察
