# 我是 OPC

[返回 Wiki 首页](../README.md)

如果你是 OPC，这个仓库最重要的价值不是“看资料”，而是帮你更快判断：

- 哪个城市已经有社区
- 哪个社区更适合你
- 进入之前应该先了解哪些工具和信息

## 建议浏览顺序

### 1. 先按城市找

- [城市入口](../cities/README.md)

先判断你关心的城市是否已经有社区，或者是否有明确线索。

### 2. 再按社区比较

- [站点目录](../sites/README.md)

看社区时，重点关注这些字段：

- 面向人群
- 社区方向
- 适合阶段
- 进入门槛
- 成本
- 优惠政策
- 是否长期开放

### 3. 再看工具和资源

- [资源库](../resources/README.md)

这里更适合在你准备行动时使用，比如：

- 想申请社区
- 想搭自己的公开页面
- 想找适合自己的日常工具

## 你最关心的问题

### 这个社区适不适合我

优先看“面向人群”“社区方向”“适合阶段”。

### 进入难不难

优先看“进入门槛”“报名方式”“是否长期开放”。

### 值不值得去

优先看“可提供内容”“优惠政策”“城市环境”。
