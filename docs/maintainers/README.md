# 我是社区主理人

[返回 Wiki 首页](../README.md)

这里就是维护入口。

## 你要做什么

### 1. 新增社区

- 用 [站点模板](../../templates/site-entry.md) 新建条目
- 放到对应城市目录
- 提交 PR 或 Issue

### 2. 更新社区

- 直接改已有条目
- 重点更新报名入口、成本、开放状态、社区额外权益
- 链接失效时优先修正

### 3. 最少要补齐这些字段

- 城市
- 类型
- 面向人群
- 社区方向
- 适合阶段
- 进入门槛
- 成本
- 报名方式
- 是否长期开放
- 社区额外权益
- 公开来源

## 你直接看这三个入口

- [站点模板](../../templates/site-entry.md)
- [站点目录字段说明](../sites/README.md)
- [站点收录 Issue 模板](../../.github/ISSUE_TEMPLATE/site_submission.md)

## 什么时候更新

- 报名入口变了就更新
- 成本变了就更新
- 社区额外权益变了就更新
- 开放状态变了就更新

城市层面的政策和政策数据源，请更新到对应城市页，不放在单个站点页。
