# 我是社区主理人

[返回 Wiki 首页](../README.md)

如果你是 OPC 社区主理人，这个仓库的目标不是“收录一次就结束”，而是让你的社区持续被更多 OPC 看见，并且让信息保持可维护。

## 你可以在这里做什么

- 新增一个社区条目
- 更新已有社区信息
- 补充优惠政策、报名方式和开放状态
- 用统一字段让 OPC 更容易判断是否适合自己

## 最低建议维护字段

一个社区条目至少建议补齐：

- 城市
- 类型
- 面向人群
- 社区方向
- 适合阶段
- 进入门槛
- 成本
- 报名方式
- 是否长期开放
- 优惠政策
- 公开来源

## 建议维护节奏

- 社区政策变化时及时更新
- 报名入口失效时立即修正
- 每月至少复核一次公开链接和核心字段

## 为什么值得维护

如果条目字段完整，OPC 才能更快判断：

- 这个社区是不是适合自己
- 是否值得报名
- 当前有没有进入机会
