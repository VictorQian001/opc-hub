# 无锡

[返回城市入口](./README.md)

无锡目前是已公开 OPC 社区动作较密集的城市之一，既有市级青年友好型社区，也有高新区和锡东新城的细分社区。

## 当前说明

无锡的公开动作比较适合继续扩展成“青年友好型社区 + 产业园社区 + 数字游民社区”的多层结构。
