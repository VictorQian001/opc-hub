# 北京

[返回城市入口](./README.md)

北京当前已收录两类不同方向的 OPC 社区条目：一个偏经开区实体社区，一个偏海淀友好社区与服务计划。

## 已知站点

- [中关村 AI 北纬社区](../sites/china/north/beijing-ai-beiwei-community.md)

## 当前说明

北京的公开动作主要集中在亦庄与海淀，适合后续继续扩展为“社区 + 友好街区 + 服务计划”的多层目录。
