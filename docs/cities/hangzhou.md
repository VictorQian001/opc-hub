# 杭州

[返回城市入口](./README.md)

杭州目前已经形成多种类型的 OPC 社区形态，包括加速社区、镇街级国际社区、AI+ 产业社区、开发者社区和生态驱动型共同体。

- [培风社](../sites/china/east/hangzhou-peifengshe.md)：杭州方向的社区示例条目，信息还比较基础。
- [Honghub 鸿鹄汇](../sites/china/east/hangzhou-honghub.md)：上城区的一人创业加速社区，偏投资驱动。

- [芯模社区](../sites/china/east/hangzhou-xinmo-community.md)：萧山钱江世纪城的研发导向社区，适合机器人和 AI+ 硬件项目。
- [青聚枢纽 AGI 青创共同体](../sites/china/east/hangzhou-qingju-agi-community.md)：上城未来数智港的生态驱动型 OPC 支持平台。

## 当前说明

杭州当前可见的几个方向：

- 上城：Honghub、青聚枢纽
- 余杭：仓前 A3、良渚数栖湾
- 萧山：芯模社区
- 西湖：魔搭社区（杭州）开发者中心

如果你是 OPC，杭州这一页已经能初步按“投资驱动 / 生态驱动 / 研发导向 / 开源开发者社区 / 政策型社区”来筛。
