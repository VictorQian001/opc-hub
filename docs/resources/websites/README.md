# 实用网站

[返回资源库](../README.md)

这个目录面向 OPC 和 OPC 社区主理人，优先收录可以直接帮助你调研、建站、收集线索、交付产品和管理日常工作的实用网站。

## 推荐分组

### 内容与研究

- [Arena AI Leaderboard](./arena-ai-leaderboard.md)：看主流模型公开排名，适合做模型横向比较。
- [ChatGPT](./chatgpt.md)：适合写初稿、做整理、做研究辅助和快速输出。
- [Notion](./notion.md)：适合搭知识库、内容库和轻量项目工作台。
- [Perplexity](./perplexity.md)：适合快速查行业、政策、竞品和工具资料。

### 建站与视觉

- [Framer](./framer.md)：适合快速做官网、活动页和个人展示页。
- [Figma](./figma.md)：适合画原型、流程和界面草稿。
- [Vercel](./vercel.md)：适合快速部署官网、文档站和产品原型。

### 收集与协作

- [Tally](./tally.md)：适合做报名、申请、调研和 waitlist 表单。
- [飞书](./feishu.md)：适合管理文档、表格、表单和线索协作。
- [n8n](./n8n.md)：适合把表单、文档、通知和 AI 串成自动化流程。

### 基础设施、调试与售卖

- [Cloudflare](./cloudflare.md)：适合做域名、DNS、CDN 和基础安全。
- [Postman](./postman.md)：适合调试 API、Webhook 和第三方接口。
- [Plausible Analytics](./plausible.md)：适合轻量查看官网和活动页转化。
- [Stripe](./stripe.md)：适合做在线收款、订阅和付款链接。
- [Gumroad](./gumroad.md)：适合售卖模板、电子资料和数字产品。
