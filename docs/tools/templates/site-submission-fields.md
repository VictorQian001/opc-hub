# 站点收录字段模板

[返回模板类工具](./README.md)

这个模板适合放到邮件、表单、飞书或 GitHub Issue 里，用于统一收集站点信息。

## 模板内容

```text
站点名称：
所在城市：
公开页面链接：
联系人 / 主理人：
站点简介（100字内）：
面向人群：
可提供内容（办公 / 活动 / 社群 / 住宿 / 工具）：
是否接受外部访问：
补充说明：
```

## 适合的使用场景

- 做站点收录
- 向城市主理人征集资料
- 补全已有目录条目
