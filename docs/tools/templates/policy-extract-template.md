# 城市政策摘录模板

[返回模板类工具](./README.md)

这个模板适合把城市公开政策、通稿或政府信息快速整理成仓库可用内容。

## 模板

```text
城市：
页面标题：
发布日期：
来源机构：
原文链接：

与 OPC 相关的核心信息：
1.
2.
3.

涉及的支持内容：
- 空间
- 资金
- 算力
- 人才
- 活动
- 场景

适合落到仓库的页面：
- 城市页
- 站点页
- 政策地图
```

## 适合场景

- 做城市页
- 做政策地图
- 整理区域观察
