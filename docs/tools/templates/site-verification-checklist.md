# 站点核验清单

[返回模板类工具](./README.md)

这个模板用于判断一个站点是否适合正式进入目录。

## 清单

```text
[ ] 是否有公开可访问链接
[ ] 是否能确认城市
[ ] 是否能确认站点名称
[ ] 是否能确认站点定位或类型
[ ] 是否能找到发布时间或最近一次公开信息
[ ] 是否能标注来源
[ ] 如果信息不完整，是否已标注“待核验”
[ ] 是否避免使用未经公开确认的隐私信息
```

## 适合场景

- 新增站点
- 校验旧条目
- PR 自查
