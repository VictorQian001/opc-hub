## 按状态理解目录

- 已收录：已有较明确的公开入口和基本信息
- 待核验：有线索，但信息还不完整
- 征集中：目录位置已预留，等待补充

## 这个目录的用途

这个目录按“国家 / 区域 / 城市 / 站点”组织 OPC 站点信息。

它的目标不是替代每个站点自己的官网，而是给出一个可浏览、可跳转、可核验的目录层，并且让 OPC 可以判断“哪个社区适合自己”。

## 站点页建议字段

站点页建议尽量补齐这些内容：

- 城市
- 状态
- 类型
- 面向人群
- 社区方向
- 适合阶段
- 进入门槛
- 成本
- 报名方式
- 是否长期开放
- 可提供内容
- 优惠政策
- 公开来源

其中前六项直接影响 OPC 是否能做判断，不建议长期空缺。

状态和字段说明详见：

- [状态定义](../resources/status-definitions.md)

## 如何新增站点

1. 复制 [站点模板](../../templates/site-entry.md)
2. 在 `sites/` 下补充原始条目
3. 再按需要补 wiki 页面或索引链接

## 如果你是主理人

- 报名入口

- 优惠政策
- 当前是否还开放
