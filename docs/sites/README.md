# 站点目录

[返回 Wiki 首页](../README.md)

这个目录按“国家 / 区域 / 城市 / 站点”组织 OPC 站点信息。

它的目标不是替代每个站点自己的官网，而是给出一个可浏览、可跳转、可核验的目录层。

## 站点状态

- 已收录：已有较明确的公开入口和基本信息
- 待核验：有线索，但信息还不完整
- 征集中：目录位置已预留，等待补充

状态说明详见：

- [状态定义](../resources/status-definitions.md)

## 如何新增站点

1. 复制 [站点模板](../../templates/site-entry.md)
2. 在 `sites/` 下补充原始条目
3. 再按需要补 wiki 页面或索引链接
